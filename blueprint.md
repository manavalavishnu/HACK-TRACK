# **App Name**: HackTrack: The Agile Sprint Tracker

## Core Features:

- Sprint Progress Tracking: Visualize sprint progress with modern, segmented progress bars.
- Roadmap Visualization: Display project roadmap with dashed lines and arrow indicators.
- Task Assignment: Assign tasks to team members.
- Live Timer: Implement a live timer for sprints with critical status indicators.
- AI-Powered Task Suggestions: Suggest new tasks based on team progress and goals using an AI tool. The AI uses reasoning to determine relevant tasks and incorporates roadmap and deadline information in its task generation.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) for innovation and speed.
- Background color: Deep Navy (#1A237E) for structure.
- Accent color: Bright Yellow (#FFEA00) for CTAs and status indicators.
- Body and headline font: 'Inter' sans-serif, suitable for a dashboard environment.
- Use icons related to speed (rockets, lightning bolts), planning (compass, map pins), and success (trophies, checkmarks).
- Use structured card layouts and subtle grid lines to represent the track and roadmap.
- Subtle animations to indicate progress and updates.
